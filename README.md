Akademi Randevu Sistemi (ARS) - README

Proje Tanımı: 
Akademi Randevu Sistemi (ARS), üniversite ortamında öğrenci ve akademisyen arasındaki iletişimi dijitalleştirmek amacıyla geliştirilen bir mobil uygulamadır. Mevcut süreçlerdeki koordinasyon sorunlarını çözmeyi hedefleyen bu platform, randevu yönetimini manuel takipten kurtararak yapay zeka destekli bir yapıya taşımaktadır.

Geliştirici Bilgileri: 
Bu proje, BOZ402 – Mobil Programlama dersi kapsamında Simay Şahiner (22040020) tarafından hazırlanmıştır.
